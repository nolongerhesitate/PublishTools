# PublishTools（发版小工具）

## 生成Version.xml更新文件

Version.xml 内容如下：

```
<?xml version="1.0" encoding="UTF-8"?>

<item>

<version>0.8.0.1</version>

<url>http://127.0.0.1/downloads/Test.zip</url>

<changelog>https://github.com/ravibpatel/AutoUpdater.NET/releases</changelog>

<mandatory>false</mandatory>

</item>
```
